# xeu baby

备份照片，上传照片，实时浏览功能的APP。主要是为保存我儿子的照片弄的。数据保存在七牛的oos。

服务端用node.js+nest.js+redis

精力有限，肯定有很多bug。

如果对你有点用可以点个star。
